# Zeiterfassung — Privacy

Datenschutzerklärung der Android-App **Zeiterfassung**
(`com.macstacker.zeiterfassung`), gehostet über GitHub Pages.

- Deutsch: https://macstacker.github.io/Zeiterfassung-privacy/privacy/
- English: https://macstacker.github.io/Zeiterfassung-privacy/privacy_en/

Die App speichert deine Daten ausschließlich lokal auf dem Gerät — keine Cloud,
kein Server, kein Tracking. Einzige Ausnahme: ein optionaler, freiwilliger
Online-Abgleich öffentlicher Feiertage (date.nager.at), bei dem keine
persönlichen Daten übertragen werden. Diese Seite dient als Datenschutz-URL für
die Google Play Console.
